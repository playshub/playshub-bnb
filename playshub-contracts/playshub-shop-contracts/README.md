# PLayhub Shop Contracts

## Features

- PurchaseItem on playshub shop
- Upgradable: EIP-7201 supported
- Pausable
- Ownable2Step
- Withdraw: Owner can withdraw money from contract

### Further

- [ ] Support multiple currency
