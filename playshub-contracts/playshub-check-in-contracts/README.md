# Check In contracts

The check-in contract is designed to support multiple currencies, allowing users to perform check-ins using various tokens, including BNB.

## Command

### Compiler

```shell
npm run compile
```

### Test

```shell
npm run test
```

### Deploy

```shell
npm run deploy:testnet
npm run deploy:opbnb
```
